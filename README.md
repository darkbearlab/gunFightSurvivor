# 羅賓漢戰記 (gunFightSurvivor)

A top-down survival shooter game built with [Phaser 3](https://phaser.io/).

## 🎮 Game Features
- **Class System**: Choose between Warrior, Archer, or Mage.
- **Squad Management**: Recruit soldiers and archers from villages to fight alongside you.
- **World Interaction**: Capture villages, destroy noble manors, and hunt wildlife.
- **Mobile Friendly**: Features virtual joystick and touch controls.
- **Dynamic Difficulty**: Challenges scale with your reputation and time survived.

## 🛠️ Project Structure
- `index.html`: Main entry point and UI structure.
- `style.css`: Game UI styling.
- `game.js`: Core game logic and Phaser implementation.
- `player.png`: Player spritesheet.

## 🚀 How to Run
Simply open `index.html` in any modern web browser.

## ⚙️ Development
This project uses **Phaser 3.60.0**. Dependencies can be managed via `npm install` (optional, as Phaser is currently loaded via CDN for simplicity).
